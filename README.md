These are the assembly steps for the drums

1.Cut both the wood and the foam into cubes with a side measure of 12cm

2. Put the piezo sensor using double sided tape and route the cable, to put it into a fixed position, tape it to the board.
<img width="357" height="476" alt="IMG_0335" src="https://github.com/user-attachments/assets/5970243e-c4dd-4d94-8cbc-5444509ebe87" />

3. Use hot silicone to stick the foam to the wood shiny face down

4. Solder the terminals from the piezo sensor to the hifi cable and protect the joints with heat shrink tubing

5. Now is a good time to test the pad either with an arduino an some basic code or with a multimeter

6. Now repeat until you have the desired amount of pads (with the arduino nano r4 you have a max of 8 analog pins)
